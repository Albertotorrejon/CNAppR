// What is CNApp 2.0

CNApp logic, now as an R package

CNApp 2.0 is an R package that implements the core analytical framework of CNApp. It processes segmented copy number data from aCGH, SNP-array, whole-exome or whole-genome sequencing to quantify CNA levels and establish associations with molecular and clinical features — without leaving your R workflow.


Released under the MIT license (2026). Source code freely available on GitHub. Based on the original CNApp tool published in eLife (2020).

// Package Functions

Core analytical modules

CNApp 2.0 exposes the three main analysis blocks of the original CNApp as callable R functions, integrable into any bioinformatics pipeline.

⭐

Re-Seg & Score

Re-segmentation · CNA Scores · Associations

Re-segment raw copy number data, compute BCS/FCS/GCS scores and run statistical tests for associations with clinical or molecular variables.

  • Sample CNA profiling
  • BCS, FCS, GCS score computation
  • Genome-wide re-segmentation
  • Variable association tests
🧬

Region Profile

Genome Region Profiling

Generate CNA region profiles, frequency maps and correlation profiles across cohorts. Identify descriptive genomic regions.

  • CNA region profiles & frequencies
  • Correlation profiles & clustering
  • T-test & Fisher test for regions
  • Region boxplots & barplots
🤖

Classifier Model

Machine Learning · randomForest

Build and evaluate a randomForest classification model from CNA-derived variables. Predict sample classes and assess model accuracy.

  • randomForest classification
  • Multi-variable input support
  • Model accuracy assessment
  • Prediction output & barplots

Array Comparative Genomic Hybridization segmented data

🔬

SNP-Array

Single Nucleotide Polymorphism array segmented data

🧪

Whole-Exome Sequencing

WES segmented output from standard bioinformatics pipelines

🖥️

Whole-Genome Sequencing

WGS segmented data for high-resolution CNA analysis
